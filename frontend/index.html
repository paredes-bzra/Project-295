TypeScript-Vorlage

Siehe TypeScript-File Source unter /src/main.ts.

Die Kompilation des TS-Codes in JavaScript erfolgt in der WebNative Extension > Build oder alternativ über die Konsole mittels npm run build im Root-Ordner (oberste Ebene, wo das package.json File liegt). Das kompilierte (übersetzte) JavaScript File ist im Anschluss unter scripts /dist/assets/index-[XYZ].js verfügbar.

Der Live Reload (automatische Aktualisierung, sobald das main.ts ändert) kann in der WebNative Extension gestartet werden.

Fügen Sie hier Ihren HTML-Code ein. Verändern Sie nur den <body>-Tag im index.html File.
