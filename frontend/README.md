# Frontend Template

## Development server

Run `npm run dev` for a dev server. Navigate to `http://localhost:5713/`. The application will automatically reload if you change any of the source files.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Further help

To get more help on the vit CLI use `npx vite --help` or go check out the [Command Line Interface reference](https://vite.dev/guide/cli) page.
